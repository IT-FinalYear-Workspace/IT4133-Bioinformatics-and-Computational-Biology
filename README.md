# üß¨ IT4133 ‚Äì Bioinformatics and Computational Biology

This repository contains the **practical sessions, notebooks, and exercises** for the course **IT4133: Bioinformatics and Computational Biology**, offered during the **4.1 semester** of the **B.Sc. (Hons) in Information Technology** program at the **University of Vavuniya**.

All practicals are implemented in **Python**, primarily using **Jupyter Notebooks**.  
The focus is on computational methods for analyzing biological data such as DNA, RNA, and proteins.

## üß≠ Course Overview

The course explores how computational techniques can be used to solve biological problems.  
Practical sessions focus on:

- Biological data representation and sequence analysis
- Genome and proteome data processing
- Phylogenetic analysis
- Pattern recognition and data visualization
- Using machine learning techniques in bioinformatics
